# CS50W - Harvard's Web Programming Course

Repositorio personal con las soluciones y proyectos del curso **CS50's Web Programming with Python and JavaScript** de Harvard University.

## Descripción

Este repositorio contiene mis trabajos, ejercicios y proyectos del curso CS50W, donde aprendí a desarrollar aplicaciones web modernas utilizando:

- **Backend**: Python con Django
- **Frontend**: HTML, CSS, JavaScript
- **Bases de datos**: SQL
- **Herramientas**: Git, GitHub, testing, y más

## Contenido

Aquí encontrarás los proyectos y tareas completadas durante el curso:

- 📚 Ejercicios de conceptos fundamentales
- 🛠️ Proyectos de aplicaciones web
- 🗄️ Trabajos con bases de datos
- 🎨 Interfaces de usuario con HTML/CSS/JavaScript

## Objetivos del Curso

- Dominar Python para desarrollo web
- Entender el patrón MVC con Django
- Desarrollar habilidades en frontend moderno
- Implementar bases de datos relacionales
- Aprender buenas prácticas de desarrollo
